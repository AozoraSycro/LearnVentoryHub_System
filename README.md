## ✨ Key Features
* **Authentication:** Secure Registration and Login system  
* **Role-Based Access Control:** Separate dashboards for **Admin** and **Staff**  
* **Inventory Management:** Full CRUD operations for products and categories  
* **POS System:** Interface for selling products and processing payments  
* **Reporting:** Sales report and transaction history  

---

## 🛠 Tech Used
* Frontend: HTML, CSS, JavaScript  
* Backend: PHP  
* Database: MySQL

## ⚙️ How to Run

### 1. Prepare Your Environment
You need a local server to run PHP and MySQL.

* Download and install XAMPP  
https://www.apachefriends.org/index.html

* Open **XAMPP Control Panel**
* Start:
  * Apache
  * MySQL

---

### 2. Set Up Project Files

1. Download the project files  
2. Extract the folder  
3. Move the folder to:

Windows
```
C:\xampp\htdocs\LearnVentoryHub
```

macOS
```
/Applications/XAMPP/htdocs/LearnVentoryHub
```

---

### 3. Initialize the Database

1. Open browser

```
http://localhost/phpmyadmin
```

2. Create database

```
pos_store
```

3. Import file

```
pos_store.sql
```

---

### 4. Launch the System

Open browser and go to

```
http://localhost/LearnVentoryHub
```
